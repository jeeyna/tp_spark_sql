Installation & vérification

Prérequis

Commandes à exécuter

# Installer PySpark

# Vérifier l'installation

Si ça bloque
Le cas le plus fréquent est l'absence de Java. Sur Mac : brew install openjdk@11. Sur Ubuntu/Debian : sudo apt install openjdk-11-jdk. Vérifiez ensuite que JAVA_HOME est bien défini.

Prise en main

Créez un fichier ventes.csv avec le contenu suivant :

date,region,produit,montant
2026-01-05,Dakar,Ordinateur,450000
2026-01-06,Thies,Telephone,120000
2026-01-07,Ziguinchor,Ordinateur,480000
2026-01-08,Dakar,Tablette,95000
2026-01-09,Saint-Louis,Telephone,110000
2026-01-10,Kaolack,Ordinateur,470000
2026-01-11,Dakar,Telephone,125000
2026-01-12,Ziguinchor,Tablette,90000
2026-01-13,Thies,Ordinateur,460000
2026-01-14,Dakar,Ordinateur,455000
2026-01-15,Kaolack,Telephone,115000
2026-01-16,Ziguinchor,Telephone,118000

Puis lancez Python (ou un notebook Jupyter) et créez votre SparkSession :

from pyspark.sql import SparkSession

spark = SparkSession.builder \
    .appName("TP_Spark_SQL") \
    .master("local[*]") \
    .getOrCreate()

# Chargement du CSV
df = spark.read.csv("ventes.csv", header=True, inferSchema=True)

df.show()
df.printSchema()
Vérification
df.show() doit afficher les 12 lignes du fichier. df.printSchema() doit montrer que montant est bien détecté comme un entier grâce à inferSchema=True.
ÉTAPE 3 15 min

Requêtes SQL

Créez d'abord une vue temporaire pour interroger le DataFrame en SQL :

df.createOrReplaceTempView("ventes")

Exercice guidé 1 — Total des ventes par région

spark.sql("""
    SELECT region, SUM(montant) AS total
    FROM ventes
    GROUP BY region
    ORDER BY total DESC
""").show()

Exercice guidé 2 — Filtrer un seul produit

spark.sql("""
    SELECT * FROM ventes
    WHERE produit = 'Ordinateur'
""").show()

ÉTAPE 4 15 min

API DataFrame

Les mêmes résultats, mais écrits avec l'API DataFrame plutôt qu'en SQL.

Équivalent de l'exercice 1

from pyspark.sql.functions import sum, desc

(df.groupBy("region")
   .agg(sum("montant").alias("total"))
   .orderBy(desc("total"))
   .show())

Équivalent de l'exercice 2

df.filter(df.produit == "Ordinateur").show()
